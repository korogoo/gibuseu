# 운영 상태

기부스는 아직 스터디 그룹에 정식 공지 전, 테스트 단계다.

- `DISCORD_WEBHOOK_URL` 시크릿은 개인 테스트 채널로 유지해라. 사용자가 "이제 운영 시작"이라고 말하면 그때 실제 스터디 채널 웹훅으로 교체해라
- `assign-teams.yml`의 `schedule` 트리거는 주석 상태로 유지해라. 같은 신호("운영 시작")가 오면 그때 주석을 풀어서 3일 주기 자동 배정을 켜라
- 운영 시작 신호를 받으면 위 두 가지를 함께 처리하고, 이 문서에서 "테스트 단계" 관련 문구를 실제 운영 상태로 정리해라
